### Desktop (> 1024px)

- Full text visible when expanded
- Icon-only when collapsed
- Touch-friendly targets

### Mobile (< 768px)
- Sidebar collapsed by default
- Overlay mode when expanded
- Auto-closes after navigation
- Large touch targets
